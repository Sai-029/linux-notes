# User Management in Linux

This folder contains Linux user management commands.

These commands are used to:
- create users
- switch users
- check logged in users
- manage permissions
- manage groups

# Notes
Linux supports multiple users.

Each user has:
- username
- password
- permissions
- groups

Root user has full control over the system.

In cloud computing, user management is important for:
- security
- access control
- server administration
