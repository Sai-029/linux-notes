# File Permissions in Linux

This folder contains Linux file permission commands.

Permissions control:
- who can read files
- who can write files
- who can execute files

# Notes

Linux uses permissions for security.

There are 3 permission types:
- read (r)
- write (w)
- execute (x)

Permissions are assigned to:
- owner
- group
- others

Example:
rwxr-xr-x

Permissions are important for:
- protecting files
- running scripts
- cloud server security
