# File Permission Commands

## ls -l
Shows file permissions.

```bash
ls -l
```

## chmod +x
Adds execute permission.

```bash
chmod +x script.sh
```

## chmod 777
Gives full permissions.

```bash
chmod 777 file.txt
```

## chmod 644
Sets read/write permissions.

```bash
chmod 644 file.txt
```

## chown
Changes file owner.

```bash
chown user file.txt
```

## chgrp
Changes group ownership.

```bash
chgrp group file.txt
```
