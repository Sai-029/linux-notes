# Process Management in Linux

This folder contains Linux process management commands.

Processes are running programs inside Linux.

These commands are used to:
- monitor processes
- stop processes
- check CPU usage
- manage background tasks

# Notes

Every running program in Linux is called a process.

Each process has:
- PID (Process ID)
- memory usage
- CPU usage

Process management helps us:
- monitor servers
- stop unwanted programs
- manage system resources

DevOps engineers use process commands daily for troubleshooting.
