# Process Management Commands

## ps
Shows running processes.

```bash
ps
```

## ps -ef
Shows all running processes.

```bash
ps -ef
```

## top
Displays live running processes.

```bash
top
```

## kill
Stops a process using PID.

```bash
kill PID
```

## jobs
Shows background jobs.

```bash
jobs
```

## bg
Runs process in background.

```bash
bg
```

## fg
Brings background process to foreground.

```bash
fg
```

## sleep
Pauses process for some time.

```bash
sleep 10
```
